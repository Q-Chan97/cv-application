# CV Application

This is a mock CV application project as part of The Odin Project's React course. This project is meant to showcase my understanding of:
   - Component based UI structure
   - Conditional rendering
   - State management
   - Keys and Props

The languages I used for this project are: 
   - React
   - CSS

Features: 
   - Name and title
   - Contact information
   - Education, Skills, and Languages
   - Personal Summary
   - Work History
